# Hướng Dẫn Thêm Hình Ảnh

## Cấu trúc thư mục

```
public/images/
├── banner/
│   ├── cafe-space-1.jpg      (Không gian quán cafe - 1920x1080px)
│   ├── cafe-coffee-2.jpg     (Cà phê và đồ uống - 1920x1080px)
│   └── cafe-ambiance-3.jpg   (Không gian thư giãn - 1920x1080px)
├── about/
│   └── cafe-about.jpg        (Hình ảnh về quán - 800x600px)
└── menu/
    ├── espresso.jpg          (Espresso - 600x600px)
    ├── cappuccino.jpg        (Cappuccino - 600x600px)
    └── matcha.jpg            (Matcha Latte - 600x600px)
```

## Nguồn ảnh miễn phí

Bạn có thể tải ảnh từ các nguồn sau:

1. **Unsplash** (https://unsplash.com)
   - Tìm kiếm: "coffee shop", "cafe interior", "coffee cup"
   - Kích thước: Tải ảnh độ phân giải cao (1920x1080px trở lên)

2. **Pexels** (https://www.pexels.com)
   - Tìm kiếm: "coffee", "cafe", "coffee shop"
   - Miễn phí sử dụng thương mại

3. **Pixabay** (https://pixabay.com)
   - Tìm kiếm: "coffee shop", "cafe"
   - Không cần ghi công

## Yêu cầu kỹ thuật

- **Banner images**: 1920x1080px (hoặc tỷ lệ 16:9)
- **About image**: 800x600px (hoặc tỷ lệ 4:3)
- **Menu images**: 600x600px (tỷ lệ 1:1)
- **Format**: JPG hoặc WebP
- **Tối ưu**: Nén ảnh trước khi upload để tăng tốc độ tải

## Cách thêm ảnh

1. Tải ảnh từ các nguồn trên
2. Đổi tên file theo đúng tên trong cấu trúc thư mục
3. Đặt vào đúng thư mục trong `public/images/`
4. Refresh trang để xem kết quả

## Lưu ý

- Đảm bảo ảnh có chất lượng tốt và phù hợp với theme cafe
- Nên sử dụng ảnh có màu sắc ấm áp, phù hợp với không gian cafe
- Tối ưu kích thước file để website load nhanh hơn
